    class quairkit.operator.gate.single_qubit_gate.H(qubits_idx=None, num_qubits=None)¶

    A collection of single-qubit Hadamard gates.

    The matrix form of such a gate is:

    \[\begin{split}H = \frac{1}{\sqrt{2}} \begin{bmatrix} 1&1\\ 1&-1 \end{bmatrix}\end{split}\]
    Parameters:¶
    qubits_idx : Iterable | int | str | None¶

    Indices of the qubits on which the gates are applied. Defaults to 'full'.

    num_qubits : int | None¶

    Total number of qubits. Defaults to None.
